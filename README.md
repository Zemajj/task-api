# Task API

REST API для управления задачами, выполненное в рамках тестового задания.

## Стек

- PHP 8.3
- Laravel 13
- SQLite
- Bruno

## Что реализовано

Реализован CRUD для сущности `Task`:

- создание задачи
- получение списка задач
- получение одной задачи
- обновление задачи
- удаление задачи

## Эндпоинты

### Получить все задачи
`GET /api/tasks`

### Получить одну задачу
`GET /api/tasks/{id}`

### Создать задачу
`POST /api/tasks`
